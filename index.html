Simple Setup, Powerful Results

Just download, unzip, and run one command. The installer handles the rest.

For Windows

C:\zenith-ai-gateway> install.bat

Double-click the `install.bat` script. It will guide you through the setup, and automatically open the dashboard in your browser when it's done.

For macOS & Linux

~/zenith-ai-gateway$ bash install.sh

Open your terminal, navigate to the folder, and run the `install.sh` script. It will handle the setup and provide you with the dashboard URL.
